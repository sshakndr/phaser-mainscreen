# UI Test Task — Main Screen Layout (Phaser 3)

## Референс
Файл `reference_screenshot.png` в этой папке.

## Задача

Сверстать главный экран игры в **Phaser 3** (веб) по референсному скриншоту.  
Все спрайты лежат в папке `sprites/`. Логику не реализовывать — только статичный layout с правильным расположением элементов.

---

## Что НЕ нужно делать

- Реализовывать клики / переходы / анимации
- Подгружать персонажей комнаты и фоновые тайлы
- Подключаться к бэкенду или игровому коду
